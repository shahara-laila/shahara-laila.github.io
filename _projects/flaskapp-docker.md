A minimal but production-ready Flask application packaged with Docker, demonstrating containerized deployment best practices for Python web services.

## Overview

This project serves as a reference implementation for deploying Flask applications in Docker containers, covering the full lifecycle from local development to containerized production deployment.

## Key Features

- Flask application with structured routing and templates
- Multi-stage Dockerfile for optimized image size
- Docker Compose configuration for local development
- Environment-based configuration management
- Health check endpoint

## Technical Stack

- **Framework:** Python 3, Flask
- **Containerization:** Docker, Docker Compose
